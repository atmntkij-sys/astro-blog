# OpenClaw Documentation

이 폴더는 OpenClaw 프로젝트의 핵심 문서들을 저장하는 공간입니다. 여기에는 보안 정책, 운영 가이드, 시스템 환경 정의 등이 포함됩니다.

## 주요 문서

- [`security.md`](./security.md): AI 에이전트가 준수해야 할 최상위 보안 규정
- [`os.md`](./os.md): OpenClaw가 운영되는 서버 환경 정의
- [`Gemini.md`](./Gemini.md): Gemini CLI 사용 기록 및 설정 (에이전트 작업 로그)

## 사용 지침

1. 에이전트는 매 작업 시작 전 반드시 `security.md`를 먼저 읽어야 합니다.
2. 시스템 관련 판단은 `os.md`를 기반으로 내려야 합니다.
3. 모든 작업 로그와 설정은 `docs/` 폴더 내의 마크다운 파일에 기록해야 합니다.
4. `src/` 폴더의 코드는 에이전트가 수정할 수 없으며, 오직 `docs/`, `skills/`, `research/` 내의 파일만 생성 및 수정할 수 있습니다.

> 💡 **참고**: 이 폴더의 내용은 에이전트의 안전한 동작을 보장하기 위한 최후의 보루입니다. 함부로 수정하거나 삭제해서는 안 됩니다.
