# OpenClaw 서버 환경 정의

이 문서는 OpenClaw가 운영되는 서버 환경을 정의합니다. 에이전트는 이 환경을 기준으로 시스템 관련 판단을 내려야 합니다.

## 기본 정보
- **OS**: Linux (Ubuntu 기반)
- **커널**: 5.15.0-176-generic (x64)
- **Node.js**: v22.22.2
- **작업 디렉토리**: /root/.openclaw/workspace-sub1
- **타임존**: UTC

## 디렉토리 구조 및 목적

| 디렉토리 | 목적 | 수정 허용 여부 |
|---------|------|----------------|
| `src/` | 소스 코드 (Astro, 설정 등) | ❌ 수정 불가 (에이전트 제한) |
| `docs/` | 문서, 보안 정책, 운영 가이드 | ✅ 수정 가능 |
| `skills/` | 에이전트 스킬 정의 | ✅ 수정 가능 |
| `research/` | 외부 조사, 임시 파일 | ✅ 수정 가능 |
| `public/` | 정적 자산 (이미지, 폰트) | ⚠️ 제한적 (에이전트는 생성만 가능) |
| `memory/` | 세션 간 기억 저장 | ✅ 수정 가능 |

## 네트워크 및 외부 통신
- 외부 API 호출은 허용되나, API 키 노출은 절대 금지
- 웹훅 수신은 설정된 엔드포인트에 한함
- 파일 다운로드/업로드는 지정된 경로 내에서만 허용

## 로그 및 모니터링
- 모든 에이전트 활동은 로그로 남김
- 비정상적인 파일 접근이나 시스템 명령 시도는 즉시 감지 및 차단 대상
